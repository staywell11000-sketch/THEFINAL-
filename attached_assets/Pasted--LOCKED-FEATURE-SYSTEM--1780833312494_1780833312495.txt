==================================================
LOCKED FEATURE SYSTEM
=====================

IMPORTANT

Do not hide unavailable features.

Locked features must remain visible throughout the CRM.

The purpose is to encourage upgrades.

==================================================
LOCKED FEATURE UI
=================

If a feature is unavailable for the current plan:

Display it in navigation.

Add:

Lock Icon

When clicked:

Show upgrade page.

Example:

Feature Locked

Upgrade to Professional Plan to access Team Management.

[Upgrade Plan]
