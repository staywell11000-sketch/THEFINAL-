==================================================
LEAD LIMIT ENFORCEMENT
======================

Before creating a lead:

Check:

organization_usage.leads_used

against

plan.max_leads_per_month

If limit reached:

Return:

HTTP 403

Message:

"Monthly lead limit reached. Upgrade your plan to continue."

==================================================
USER LIMIT ENFORCEMENT
======================

Before inviting a user:

Check:

users_used

against

max_users

If limit exceeded:

Block creation.

==================================================
WHATSAPP LIMIT ENFORCEMENT
==========================

Before connecting WhatsApp:

Check:

whatsapp_accounts_used

against

max_whatsapp_accounts

If exceeded:

Block connection.

==================================================
FACEBOOK LIMIT ENFORCEMENT
==========================

Before connecting Facebook:

Check:

facebook_accounts_used

against

max_facebook_accounts

If exceeded:

Block connection.

==================================================
DOCUMENT STORAGE LIMIT ENFORCEMENT
==================================

Before uploading files:

Calculate:

current_storage + upload_size

If greater than:

max_storage_gb

Reject upload.
